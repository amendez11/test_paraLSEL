\begin{DoxyEnumerate}
\item La transición inválida final es \{-\/1, NULL, -\/1, NULL\}
\item Cualquier otra transición debe tener un estado válido de entrada y salida
\item Las funciones de guarda y actualización pueden ser NULL
\item Puede haber varias máquinas de estados simultáneamente sin que afecte una a otra si no tienen conexión externa
\item Si hay datos erróneos no debe reservar la memoria
\item Si falla en la reserva, devuelve NULL
\item Si La funcion de inicialización falla o hay datos erróneos, devuelve false
\item La memoria pasada no es NULL (para la estática)
\item La tabla de transiciones no es NULL
\item Todos los estados son naturales e inferiores al estado máximo
\item El estado inicial es el estado de origen de la primera transición
\item Solo se comprueban las funciones de guarda de las transiciones con estado de origen igual al de la máquina de estados
\item Si para una transición a comprobar tiene una función de guarda NULL, se cumple siempre
\item Si para una transición a comprobar tiene una función de guarda, se ejecuta la función y si devuelve distinto de 0, se cumple.
\item Si se cumple, se cambia el estado actual al de destino, no se comprueban más transiciones y se ejecuta la función de actualización, si es distinto de NULL y devuelve 1
\item Si ninguna transición tiene un estado de origen coincidente con el estado actual, la función de comprobación de transiciones devuelve -\/1.
\item Si hay transiciones pero no se cumplen las condiciones, devuelve 0 
\end{DoxyEnumerate}
